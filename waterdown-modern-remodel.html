Waterdown Modern Remodel

In Development

Summary

This modern home remodel originates from a bungalow century home. The home will be stripped to the studs and a second floor will be built above along with extensive renovations to the main floor and the exterior.

Project Features

This lot is very spacious including a large front yard and side yard which posed a unique opportunity to include many windows as privacy would not be a concern with the large yards and vegetation surrounding the lot. Along with the large modern windows, this home has been transformed with long vertical and horizontal lines to create an improved modern design. Along with the aesthetics of this home, the interior layout is functional, spacious and includes four bedrooms (including the master) with four walk-in closets and ensuites.
